### Shrnutí support ticketu

1. **Téma ticketu:**
   - Uživatel se potýkal s chybou „Data parameter Company - bad format“ při používání API.

2. **Spokojenost:**
   - Pravděpodobně spokojený – uživatel potvrdil funkčnost.

3. **Klíčové kroky provedené během řešení:**
   - uživatel byl požádán o zaslání kódu API dotazu.
   - zjistilo se, že pro správnou funkčnost je potřeba používat ID společnosti místo jejího názvu.
   - uživatel byl informován o tom, jak najít a použít ID společnosti.
   - po poskytnutí správných informací uživatel potvrdil, že problém byl vyřešen.

4. **Poznámky:**
   - řešení ticketu trvalo 1162 minut.
   - zahrnovalo šest komentářů, z toho tři od uživatele a tři od supportu.
   - uživatel musel projít procesem učení, jak používat API správně.
   - komunikace byla efektivní a přátelská.

5. **Doporučení pro vývoj/support:**
   - zvážit možnost vývoje podrobnější dokumentace k API pro uživatele.
   - přidat postup pro získání ID společnosti přímo do systému, aby se předešlo podobným problémům.
   - zajistit příznivější viditelnost potřebných informací pro nové uživatele API.
