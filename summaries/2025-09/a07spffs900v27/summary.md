### Shrnutí support ticketu

1. **Téma ticketu:**
   - Uživatel potřeboval poradit ohledně API endpointu pro získání seznamu úkolů a detailů úkolu.

2. **Spokojenost:**
   - Pravděpodobně spokojený – uživatel potvrdil, že mu byla poskytnuta potřebná pomoc.

3. **Klíčové kroky provedené během řešení:**
   - uživatelovi byla vysvětlena funkce atributu "section" a jak jej získat.
   - doporučena struktura API dotazů pro získání potřebných dat.
   - informováno o možnostech použití pohledu pro filtrování úkolů.

4. **Poznámky:**
   - Řešení trvalo přibližně 7 dní.
   - Během komunikace bylo vyměněno 7 komentářů.
   - Celý proces byl veden online formou.
   - Uživatel byl pouze jeden a interakce byla převážně od jednoho agenta.

5. **Doporučení pro vývoj/support:**
   - Zvážit úpravy dokumentace API pro lepší přehlednost atributů.
   - Vytvořit tutoriály pro komplexnější použití API.
   - Umožnit uživatelům snadnější přístup k předdefinovaným dotazům na úkoly.
